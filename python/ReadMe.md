yamlscript
==========

Program in YAML


## Synopsis
```
#!/usr/bin/env yamlscript

defn main(name):
  say: "Hello, $name!"
```


## Status

This module is *very* **ALPHA**.

YAMLScript is still being defined.

Use with caution for now.


## Description

YAMLScript is a programming language encoded in YAML.
